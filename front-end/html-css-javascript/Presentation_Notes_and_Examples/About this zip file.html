This zip file contains all the presentation notes (in PDF format) and examples developed for the MOOC course 'HTML, CSS and JavaScript' by Prof. David Rossiter of the Hong Kong University of Science & Technology. This course is the first in a series of courses which comprise the 'Full Stack Web Development' Specialization.

There are many folders in this zip file, reflecting the structure of the course. After you extract everything to your local storage selecting a link in one of the PDF files will open the example in the same directory (not from the web).

Hope you find these useful!
